
# Propriété File.size
Obtient la taille du fichier de document en octets.

|||
|:-----|:-----|
|**Hôtes :**|PowerPoint,Word|
|**Ajouté dans**|1.1|

```
var fileSize = file.size;
```


## Valeur renvoyée

Taille du fichier en octets.
