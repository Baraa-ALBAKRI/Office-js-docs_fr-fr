
# DisplayName, élément
Spécifie le nom de votre complément Office sous la forme d’une chaîne de 125 caractères maximum.

 **Type de complément :** Application de contenu, de volet Office, de messagerie


## Syntaxe :


```XML
<DisplayName DefaultValue="string ">
```
