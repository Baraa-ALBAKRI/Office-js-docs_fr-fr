
# TabletSettings, élément
Spécifie les paramètres de contrôle qui s’appliquent lorsque votre complément de messagerie est utilisé sur une tablette.

 **Type de complément :** messagerie


## Syntaxe :


```XML
<TabletSettings>
   ...
</TabletSettings>
```
