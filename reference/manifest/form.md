
# Élément Form
Paramètres UX pour les formulaires que votre complément de messagerie utilisera lors de l’exécution sur un appareil particulier (ordinateur de bureau, tablette ou téléphone).

 **Type de complément :** messagerie


## Syntaxe :


```XML
<Form>
   ...
</Form>
```
